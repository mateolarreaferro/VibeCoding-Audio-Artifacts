the logistic map is x(n+1) = r·x(n)·(1-x(n)). different r values create different behaviors. below 3.0 you get stable tones. around 3.0-3.57 there's period doubling. above 3.57 it goes chaotic. moving the mouse left to right lets you explore these different states in real time.

the synthesis uses 5 oscillators with slight detuning and stereo reverb. everything updates smoothly at 48khz with envelope ramping to avoid clicks. three processes run in parallel: one reads mouse input, one updates the chaos, and one controls the oscillators.

i'm thinking of this as exploring the boundary between control and unpredictability. you set the parameters but the logistic map generates the actual patterns. it works well as an improvisation tool.
